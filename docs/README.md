# 🎬 Cinespoilers API

Aplicación desarrollada en Django Rest Framework para gestionar películas en un sistema de cine mediante operaciones CRUD.

---

## Tecnologías utilizadas

* Python
* Django
* Django Rest Framework
* SQLite
* Postman
* Git & GitHub
